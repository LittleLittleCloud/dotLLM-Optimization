# Benchmark Results

This directory stores benchmark output files in `.jsonl` format (one JSON object per line).

## Output Format

Each line is a JSON object with these fields:

| Field | Type | Description |
|-------|------|-------------|
| `engine` | string | Engine name (e.g., "ollama", "vllm", "dotllm") |
| `model` | string | Model identifier |
| `quantization` | string | Quantization type (e.g., "int4") |
| `prompt_name` | string | Prompt category: "short", "medium", "long" |
| `input_tokens` | int | Number of input tokens |
| `output_tokens` | int | Number of generated tokens |
| `time_to_first_token_ms` | float | Time to first token in milliseconds |
| `prefill_tokens_per_sec` | float | Prefill throughput (input_tokens / TTFT) |
| `decode_tokens_per_sec` | float | Decode throughput (output_tokens / decode_time) |
| `total_time_sec` | float | Total generation time in seconds |
| `run_index` | int | Run number (for averaging multiple runs) |
| `timestamp` | string | When the benchmark was run |
| `notes` | string? | Optional notes |

## Files

- `{engine_name}.jsonl` — Raw results per engine
- `comparison.csv` — Aggregated comparison (generated by `analyze.py`)
